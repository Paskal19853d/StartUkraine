## Опис проекту

**Зоряна Пам'ять** — це український меморіальний веб-додаток, присвячений вшануванню пам'яті загиблих під час російського вторгнення в Україну. Це інтерактивна карта-база даних, де користувачі можуть додавати, шукати та вшановувати пам'ять полеглих воїнів і мирних жителів — із фотографіями, біографічними відомостями та особистими історіями.

**Репозиторій:** https://github.com/Paskal19853d/StartUkraine.git

## Технологічний стек

### Frontend
- HTML5 з вбудованим SVG (детальна карта України)
- CSS3 (913 рядків) з CSS-змінними для тематизації
- JavaScript — анімації, логіка карти, пошук
- WebGL — інтерактивна флюїдна симуляція фону (Pavel Dobryakov, MIT license)
- Шрифти Google: Geologica (текст), Unbounded (заголовки)
- Без зовнішніх JS-фреймворків (чистий Vanilla JS)

### Backend
- Python 3 + **FastAPI**
- **SQLite3** — база даних (`memorial.db`)
- CORS Middleware
- WebSocket — онлайн-лічильник користувачів в реальному часі
- Запуск: `python -m uvicorn Paskal:app --reload --port 8000`
- Статичні файли через `FastAPI StaticFiles`
- PHP-проксі (`proxy.php`) для інтеграції з alerts.in.ua

## Функціональність

### Публічний інтерфейс (`index.html`)
1. **Інтерактивна карта** — зум/панорама SVG-карти України; маркери меморіалів на координатах
2. **Пошукова система** — нечіткий пошук з транслітерацією (кирилиця ↔ латиниця); пошук за ім'ям, позивним, місцем; зважений скоринг (прізвище 2.0x, ім'я 1.8x)
3. **Картки меморіалів** — повна біографічна інформація, лайки, обставини загибелі
4. **Автентифікація** — реєстрація/вхід з модальним вікном
5. **Додавання меморіалу** — форма з модерацією; вибір кольору маркера
6. **WebGL-ефекти** — флюїдна симуляція, що реагує на курсор
7. **Онлайн-лічильник** — WebSocket, оновлення в реальному часі

### Адмін-панель (`admin.html`)
- Логін: `admin@admin.com` / `Admin`
1. **Статистика** — загальна кількість записів, користувачів, лайків, онлайн
2. **Записи** — повне CRUD-управління меморіалами
3. **Користувачі** — бан/розбан
4. **Кольори** — редактор теми в реальному часі
5. **Редактор карти** — перетягування підписів областей, налаштування кольору і розміру

## Алгоритм пошуку (`Paskal.py`, рядки 350–464)

- Нормалізація тексту: нижній регістр + транслітерація
- Нечіткий скоринг з кількома пріоритетами
- Зважені поля: прізвище → ім'я → позивний → місце загибелі → місце поховання
- Повний збіг імені — найвищий пріоритет
- Аналітика: усі запити пишуться в `search_logs`

## Висновок

**Зоряна Пам'ять** — це технічно зрілий меморіальний додаток з продуманою архітектурою та значущим змістом. Сильні сторони проекту:

- Підтримка Української мови (транслітерація у пошуку)
- Потужний FastAPI-бекенд зі структурованою схемою даних
- WebGL-ефекти без залежностей від зовнішніх фреймворків
- Дворівнева модерація для забезпечення якості даних
- Адмін-панель із повним управлінням контентом і темою

**Потенційні покращення:**
- Замінити SHA256 на bcrypt/argon2 для паролів
- Додати FTS5-індекси SQLite для прискорення пошуку
- JWT-токени замість передачі email/password в адмін-запитах
- Пагінація для `/api/people` (522+ записів)
- Обробка помилки бічної панелі (активна задача)
